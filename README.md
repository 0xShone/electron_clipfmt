# electron_clipfmt

## About this
*Watching clipboard* and *Converting* `full-width letter` -> `half-width letter`

## Run
```bash
cd <this repo>
npm install
npm run build
npm start
```

You can also run app with
- `<this repo>/app/clipfmt-darwin-x64/clipfmt.app` (Mac)
- `<this repo>/app/clipfmt-win32-x64/clipfmt.exe`  (Windows)
